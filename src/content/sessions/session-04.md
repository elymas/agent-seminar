## 미션 설명

자신의 업무에 최적화된 컨텍스트 파일 세트를 작성합니다.

## 실습 가이드

1. about-me.md 작성 (역할, 핵심 업무, 선호하는 결과물 형식)
2. working-rules.md 작성 (AI와의 협업 규칙)
3. 세션 2에서 만든 프롬프트를 컨텍스트와 함께 업그레이드

## 템플릿: about-me.md

```
# About Me

- 나는 [직책]으로 [부서]에서 일합니다
- 주요 업무는 [업무1], [업무2], [업무3]입니다
- 선호하는 결과물 형식: [간결한 표 / 상세 보고서 / 불릿 포인트]
- 자주 다루는 클라이언트/프로젝트: [여기에 입력]
```

## 템플릿: working-rules.md

```
# Working Rules

- 불확실한 정보는 추측하지 말고 '확인 필요'로 표시하세요
- 결과물은 반드시 한국어로 작성하세요
- 수치 데이터를 포함할 때는 출처를 명시하세요
- 보고서는 [형식]을 따라주세요
```

## 제출

about-me.md와 working-rules.md를 공유 폴더에 제출해주세요.
