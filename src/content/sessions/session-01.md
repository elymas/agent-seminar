## 핵심 메시지

Claude는 Chat, Projects, Cowork, Claude Code 등 여러 "표면"이 있고, 각각 다른 종류의 일에 적합합니다. 챗봇처럼 질문하는 것(Level 1)과 에이전트로 운영하는 것(Level 5)은 완전히 다른 경험입니다.

## 데모

같은 업무를 챗봇 방식 vs 에이전트 방식으로 처리하는 비교 시연

- **챗봇 방식:** "경쟁사 동향 정리해줘" → 일회성 응답
- **에이전트 방식:** CLAUDE.md + 구조화된 프롬프트 → 일관된 브리프 자동 생성
