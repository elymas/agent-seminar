## 미션 설명

자신의 반복 업무 1개를 Skill로 설계합니다.

## 실습 가이드

1. 매주 반복하는 업무 중 가장 시간이 많이 걸리는 것 1개 선택
2. SKILL.md 형식으로 작성
3. 동료와 교차 검증 (이 설명으로 AI가 실행할 수 있는가?)

## 템플릿: SKILL.md

```
---
name: [스킬 이름]
description: [이 스킬이 무엇을 하는지, 언제 사용하는지]
---

# Objective

[이 스킬의 목표를 한 문장으로]

# Inputs

- [입력 1]: [설명]
- [입력 2]: [설명]

# Process

1. [단계 1]
2. [단계 2]
3. [단계 3]
4. [단계 4]
5. [결과물 저장]

# Gotchas

- [주의사항 1]
- [주의사항 2]
- 불확실한 정보는 추측하지 말고 명시적으로 표시할 것
```

## 체크리스트

- [ ] 다른 사람이 이 설명만 보고 같은 작업을 할 수 있는가?
- [ ] 예외 상황이 빠지지 않았는가?
- [ ] 입력과 출력이 명확한가?
