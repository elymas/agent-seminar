## 미션 설명

자신의 핵심 업무 3가지를 구조화하여 AI에게 설명하는 프롬프트를 작성합니다.

## 실습 가이드

1. 아래 템플릿에 나의 역할, 핵심 업무, 자주 하는 반복 작업을 채워넣으세요
2. 가장 시간이 많이 걸리는 업무 1개를 골라 Claude에게 설명해보세요
3. Claude의 응답 품질을 평가하고 프롬프트를 개선해보세요

## 템플릿

```
# 나의 업무 프로필

## 역할
- 직책: [여기에 입력]
- 소속: [여기에 입력]

## 핵심 업무 (상위 3가지)
1. [업무 1]
2. [업무 2]
3. [업무 3]

## 가장 시간이 많이 걸리는 작업
- 작업명: [여기에 입력]
- 빈도: 매주 / 매일 / 매월
- 소요 시간: 약 [X]시간
- 현재 방식: [어떻게 하고 있는지]
```

## 제출

작성한 프롬프트를 공유 폴더에 제출해주세요. 미완성 시 세션 3 전까지 과제로 완성합니다.
