## Typography

| Role | Size | Weight | Line Height | Font |
|------|------|--------|-------------|------|
| Body | 16px | 400 (regular) | 1.5 | Pretendard (font-sans) |
| Label / micro | 10px → `--text-label` | 400 (regular) | 1 | IBM Plex Mono (font-mono) |
| Secondary label | 12px (text-xs) | 600 (semibold) | 1 | IBM Plex Mono (font-mono) |
| Heading | 20–28px (context-dependent) | 600 (semibold) | 1.2 | IBM Plex Mono (font-mono) |

**Two declared weights: 400 (regular) and 600 (semibold).** No third weight is permitted in this phase.

Note on existing `font-bold` (700) usages: The codebase currently contains `font-bold` on page `h1`/`h2` headings, metadata value cells in `[...slug].astro` (week number, time, deliverable), and accent-colored step-number badges across multiple pages. None of these elements are touched in Phase 1. Normalization of `font-bold` to `font-semibold` across the full codebase is deferred to a later phase. Phase 1's Implementation Checklist below modifies only `font-medium` (500) → `font-semibold` (600) at line 147 of `[...slug].astro`.

### New Token: `--text-label`

**Value:** `10px`
**Replaces:** All 14 instances of `text-[10px]` in the codebase:
- `src/pages/sessions/[...slug].astro` — 10 instances (Session Progress labels, metadata section labels, TOC label, prev/next navigation labels)
- `src/components/AdoptionLadder.astro` — 4 instances (level description and week labels, both desktop and mobile layouts)

**After token is declared**, all `text-[10px]` classes are replaced with `text-label`.

### Font Weight Normalization: TYP-02

**Location:** `src/pages/sessions/[...slug].astro` line 147, TOC depth-2 heading links
**Before:** `text-text-secondary font-medium`
**After:** `text-text-secondary font-semibold`
**Rationale:** `font-medium` (500) is the sole outlier in a system that uses only `regular (400)` and `semibold (600)`. Normalized to maintain the two-weight constraint.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#0a0a0a` (`--color-bg`) | Page backgrounds |
| Secondary (30%) | `#111827` (`--color-bg-card`) | Cards, sidebar, session detail metadata panel |
| Accent (10%) | `#10b981` (`--color-accent`) | See reserved list below |
| Destructive | none | No destructive actions in this phase |

Accent reserved for: progress bar current-session indicator, practice-type badge text, active/current state highlights, CTA buttons. Not used for concept or special type badges.

Source: `CLAUDE.md` color section + `src/styles/global.css` `@theme` block.

### New Tokens: CLR-01 — Session Type Badge Colors

The badge color system for session type (`concept`, `practice`, `special`) currently uses Tailwind's built-in color names (`blue-500`, `yellow-500`). These are opaque to the design system. Two semantic tokens are added to `@theme`:

| Token | Value | Usage |
|-------|-------|-------|
| `--color-badge-concept` | `#3b82f6` | Background tint and text for concept-type badges (rgb 59, 130, 246 — Tailwind blue-500) |
| `--color-badge-concept-bg` | `rgba(59, 130, 246, 0.10)` | 10% tint background for concept badge |
| `--color-badge-concept-border` | `rgba(59, 130, 246, 0.20)` | 20% tint border for concept badge |
| `--color-badge-special` | `#eab308` | Background tint and text for special-type badges (rgb 234, 179, 8 — Tailwind yellow-500) |
| `--color-badge-special-bg` | `rgba(234, 179, 8, 0.10)` | 10% tint background for special badge |
| `--color-badge-special-border` | `rgba(234, 179, 8, 0.20)` | 20% tint border for special badge |

`practice` badges already use `bg-accent/10`, `text-accent`, `border-accent/20` — these reference existing `@theme` tokens and require no change.

**Files affected:**
- `src/pages/sessions/[...slug].astro` — `typeColor` record (lines 28–31)
- `src/components/SessionCard.astro` — inline ternary badge class (line 37)

### New Token: CLR-02 — Blockquote Background

**Location:** `src/pages/sessions/[...slug].astro` inline `<style>` block, line 314
**Before:** `background: rgba(16, 185, 129, 0.05);`
**After:** `background: var(--color-accent-subtle);`

| Token | Value | Usage |
|-------|-------|-------|
| `--color-accent-subtle` | `rgba(16, 185, 129, 0.05)` | Session detail blockquote callout background — 5% accent tint |

This is the only location in the codebase that uses a 5% accent tint. No other files reference this value.
